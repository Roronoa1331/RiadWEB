# Personal Website

A minimalist static personal website inspired by https://mhouge.dk/.

## Files

- `index.html` — main homepage markup
- `styles.css` — minimalist layout and typography

## How to use

Open `index.html` in your browser, or serve the folder with a simple static server.

## Customize

Replace placeholder text and links with your name, email, projects, blog posts, and social profiles.
